## 💡 How to Use GeoLuminamics

### Creating Your First Light Animation

1. **Open the app.** You will see a blank canvas with some tool options on the left.

2. **Add shapes.** Use the shape tool to place basic geometry like circles, squares, and polygons.

3. **Adjust light properties.** Select each shape and choose color, brightness, and movement options using the control panel.

4. **Animate.** Use the timeline at the bottom to set when lights appear and move. Press play to preview your animation.

5. **Save your work.** Use File > Save to keep your project. You can open it later to continue editing.

### Tips for Smooth Flow

- Start simple: Use a few shapes and colors before adding complex designs.
- Experiment with different light speeds and patterns.
- Use the undo button if you do not like a change.
- Check the preview often to see how your animation looks.
